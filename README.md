# Vconsole: advanced console

- [x] Typescript
- [x] Coffeescript
- [x] LiveScript

### Embed
```js
// 2 Params: code - language
const code = encodeURIComponent(btoa("console.log('hello')"));
const language = 'livescript';

// -> https://vconsole.vercel.app?language=language&code=code
```
